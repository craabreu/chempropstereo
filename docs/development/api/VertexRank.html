VertexRank¶

class chempropstereo.VertexRank(value)[source]¶

Enumeration of vertex ranks for tetrahedral stereochemistry.

Variables:

  • NONE (int) – Not a tetrahedral vertex.

  • FIRST (int) – The first vertex in canonical order.

  • SECOND (int) – The second vertex in canonical order.

  • THIRD (int) – The third vertex in canonical order.

  • FOURTH (int) – The fourth vertex in canonical order.

Examples

>>> from rdkit import Chem
>>> from chempropstereo import stereochemistry
>>> mol = Chem.MolFromSmiles("C[C@H](N)O")
>>> stereochemistry.tag_tetrahedral_stereocenters(mol)
>>> bond = mol.GetBondWithIdx(1)
>>> VertexRank.from_bond(bond)
<VertexRank.SECOND: 2>
>>> VertexRank.from_bond(bond, end_is_center=True)
<VertexRank.NONE: 0>
